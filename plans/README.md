# Plans Archive

Store approved Claude Code plans here before executing them.

## Why
- Lets you revert specific features by feeding the original plan back as context
- More precise than reverting git commits
- Builds a history of architectural decisions

## Naming Convention
YYYY-MM-DD_feature-name.md

## Workflow
1. Enter plan mode (Shift+Tab)
2. Review and approve the plan
3. Save it here as a .md file
4. Clear context (Ctrl+R or /clear)
5. Run with auto-accept enabled
