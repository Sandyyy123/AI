# Claude Code Failure Tracker

Track tasks the model struggles with. Re-test after each model update.

| Date | Model | Task Description | What Failed | Status | Re-test Date |
|------|-------|-----------------|-------------|--------|--------------|
| | | | | | |

## How to Use
- Add a row whenever Claude fails a task or produces poor output
- Note the specific model version (e.g. claude-sonnet-4-6)
- After a new model drops, go through this list and re-test each item
- Update status to "resolved" if it now works, or keep tracking

## Why This Matters
Models evolve fast. Tasks with 30% success rate today may hit 90% next month.
This list ensures you never give up on a capability too early.
